2. 매주 일요일 `11:00` 코드 리뷰
3. **각 주차별 Directory Name** : 'Week{num}'
    - num : 2자리 숫자로
    - ex) Week01
4. **Directory Name** : `문제번호`
    - 백준 : 2573
    - 프로그래머스 : 문제명
5. **File Name** : `문제번호_이름`  
    - 백준 : 2573_sm.java  
    - 프로그래머스 : 문제명_sm.java
6. **Commit Message** : `[Solve] ✍ 사이트명_문제번호.문제명`
